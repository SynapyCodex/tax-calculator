# โปรแกรมคำนวณภาษีเงินได้บุคคลธรรมดา

โปรแกรมคำนวณภาษีเงินได้บุคคลธรรมดาแบบง่ายๆ พัฒนาด้วย HTML, CSS และ JavaScript

## ฟีเจอร์

- ✅ คำนวณภาษีแบบขั้นบันได (0% - 35%)
- ✅ แสดงกราฟภาษีแบบ Animation
- ✅ UI สวยงาม พร้อม Background Animation
- ✅ Responsive Design รองรับทุกอุปกรณ์
- ✅ แจ้งเตือนเมื่อไม่ได้กรอกข้อมูล

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์
2. กรอกข้อมูล:
   - เงินได้ทั้งปี
   - หักค่าใช้จ่าย
   - ค่าลดหย่อน
3. กดปุ่ม "คำนวณภาษี"
4. ดูผลการคำนวณและกราฟ

## อัตราภาษี

- 0% สำหรับรายได้สุทธิไม่เกิน 150,000 บาท
- 5% สำหรับรายได้ 150,001-300,000 บาท
- 10% สำหรับรายได้ 300,001-500,000 บาท
- 15% สำหรับรายได้ 500,001-750,000 บาท
- 20% สำหรับรายได้ 750,001-1,000,000 บาท
- 25% สำหรับรายได้ 1,000,001-2,000,000 บาท
- 30% สำหรับรายได้ 2,000,001-5,000,000 บาท
- 35% สำหรับรายได้เกิน 5,000,000 บาท

## เทคโนโลยีที่ใช้

- HTML5
- CSS3 (Animations, Gradients)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Kanit)

## หมายเหตุ

โปรแกรมนี้เป็นแบบง่ายๆ สำหรับการศึกษาเท่านั้น การคำนวณภาษีจริงอาจมีรายละเอียดเพิ่มเติม
